computational scAAVengr

1. get cell atlas (Arc Virtual Cell Atlas + State)
    > pull multi-species cell data, specific organ data
2. learn cell embeddings (scVI)
    > numeric vectors for each cell type
3. get capsid embeddings (Evo2)
    > numeric vectors from capsid amino acid sequences
4. encode promoters (learned embeddings)
    > vector for each promoter type
5. predict tropism (tropism ML head)
    > combine cell + capsid + promoter vectors -> transduction rates
6. check stability (Rosetta)
    > filter out capsids with bad biophysics
7. generate synthetic scRNA-seq off of species * organ * state (scVI.generate)
    > make realistic read counts from predicted rates
8. build reference (barcode-augmented FASTA)
    > add barcodes for alignment
9. align reads (STARsolo)
    > produce cell x gene count matrix
10. analyze (Scanpy + scVI)
    > clustering, UMAP, metrics
11. explore (CellxGene)
    > create interactive visualization and sharing
