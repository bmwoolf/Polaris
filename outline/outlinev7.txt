customer: scientist
goal: predict AAV capsid x promoter transduction / specific cells

> get cell data
    - Arc Virtual Cell Atlas (State)- retina + brain cell profiles with annotations 
> make cell embeddings 
    - scVI- probabilistic latent space for all cell types 
        - we might be able to do this with State itself, since we perturb the cell when we deliver the RNA in the virus
> get capsid data
    - Evo2- embeddings from AAV viral protein sequences (AAv2, AAV6, DJ, Anc80)
    - ESM-2- tokenize amino acids -> per-residue states -> mean/attn pool -> capsid vector
> encode promoters
    - strength metadata preloaded, one-hot vectors for CAG, CMV, RHO
> predict tropism
    - PyTorch MLP- input (cell embedding, capsid embedding, promoter vector) -> output % expression
    - ingest input concatenated vector of [cell_embedding, capsid_embedding(Evo2+ESM-2), promoter_vector]
> simulate scRNA-seq
    - scDesign3- generate synthetic counts conditioned on predicted transduction + barcode integration
> build reference genome
    - custom FASTA script- append 8nt barcodes to genome for mapping
> align + analyze
    - STARsolo -> ScanPy alignment, QC, clustering, UMAP, CellType x Serotype heatmaps
> share
    - CellxGene
